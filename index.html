Q-1 : What is ol,ul and li in html?

Ans : The ol tag stands for ordered list It is used to create a numbered list, where each item is automatically assigned a number.The ul tag stands for unordered list. It is used to create a bulleted list, where each item is preceded by a bullet point.The litag stands for list item.It is used to define an individual item within a list, whether it's an ordered list or an unordered list.

Ans :td represents a standard data cell within a table. It is used to contain regular content or data in a table row. th represents a header cell within a table. It is used to define the header for a column or a row in a table.
